## Projects

### [NACA Airfoil Surrogate Model](https://github.com/miguelRepo/naca-surrogate)
A Random Forest surrogate model trained on 20,000+ XFoil aerodynamic simulations.
Predicts lift and drag coefficients for NACA 4-digit airfoils instantly from geometry 
and flight conditions.  
**[Live demo →](https://naca-airfoil-surrogate.streamlit.app)**  
`Python` `XFoil` `scikit-learn` `Streamlit`

### [US Aviation Accident Analysis (2008–2025)](https://github.com/miguelRepo/aircraft-accidents-analysis)
Exploratory analysis of 18,498 NTSB aviation accident records. Covers accident trends, 
phase-of-flight risk, human factors, and IMC vs VMC fatality rates.  
`Python` `Pandas` `Matplotlib` `SQLite`
